# Задание 1

### Текст задачи
На основе списка вещественных чисел получить список целых чисел, отбросив дробную часть.

### Алгоритм решения
Функция запроса вещественного числа: 
На вход запрашивается вещественное число и возвращается.

Главная функция:
На вход запрашивается количество чисел, на основе которых многократно вызывается функция запроса вещественного числа.
Полученные числа добавляются в список.
Создается второй список, значениями которого являются целые части элементов (отбрасывается дробная часть) из первого списка. 
Полученные списки выводятся на экран.

# Задание 2

### Текст задачи
Список содержит строки. Сколько из них имеет чётную длину?

### Алгоритм решения
Функция запроса строки:
На вход запрашивается строка и возвращается.

Главная функция:
На вход запрашивается количество строк, на основе которых многократно вызывается функция запроса строки.
Полученные строки добавляются в список.
У каждого строки списка считается длина и проверяется на четность. Если длина строки четная, то количество четных чисел увеличивается на единицу, иначе не изменяется.
Список строк и количество четных длин строк выводятся на экран.
